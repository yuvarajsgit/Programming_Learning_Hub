# Programming Learning Hub 👨‍💻

A clean, structured web platform designed to streamline student registration and provide quick access to core programming languages, software engineering concepts, and curated learning tracks. 

The platform offers an intuitive entryway for beginners to understand logical workflows and academic computer science domains before transitioning directly to study resources.

##  Live Links
* **Repository Link:** [https://github.com/yuvarajsgit/programming_learning_hub](https://github.com/yuvarajsgit/programming_learning_hub)

---

##  Tech Stack & Features

* **Markup & Structure:** Semantic HTML5 (`index.html` and `registration.html`) modeling localized user onboarding routing.
* **Layout Mechanics:** Structured multi-column informational mapping using HTML data matrix matrices (tables).
* **Cross-linking:** Seamless index-to-registration local pathway navigation links.

---

##  Core Architecture & Page Breakdowns

### 1. Home Dashboard (`index.html`)
Serves as the information architecture hub, breaking down the essential conceptual blocks of software engine engineering.
* **Core Concepts Covered:** Maps out the complete structural breakdown of software design, including *Problem Definition, Algorithms, Flowcharts, Logic Optimization, Testing, Debugging,* and *Documentation*.
* **Tech-Stack Matrix:** Features a comprehensive reference grid categorizing 50 foundational industry tools across 10 critical domains:
  * **Languages:** C, C++, Java, Python, JavaScript
  * **IDE/Environments:** VS Code, Eclipse, IntelliJ, PyCharm, NetBeans
  * **Web Technologies:** HTML, CSS, JavaScript, React, Angular
  * **Databases:** MySQL, PostgreSQL, MongoDB, Oracle, SQLite
  * **DevOps & Testing:** Docker, Kubernetes, Jenkins, Git/GitHub, JUnit, Selenium, Cypress

### 2. Validated Onboarding Gateway (`registration.html`)
An interactive intake form containing complete programmatic selection fields to group prospective student demographics:
* **Demographic Input Controls:** Personal identification tracking nodes (Name, Age, Date of Birth selection, Radio-grouped Gender elements).
* **Academic Profiling:** Department sorting dropdown mapping fields spanning CSE, IT, ECE, EEE, MECH, CIVIL, BME, BTech, AIDS, and AIML.
* **Target Competency Filters:** Custom-tailored multi-tiered parameters allowing users to route their profile by target language, skill hierarchy tier (*Beginner, Intermediate, Advanced*), and educational tier classification (*High School, Diploma, UG, PG, PhD*).
* **Document Upload & Rules Compliance:** Built-in structural fields for resume ingestion (`type="file"`) along with logical user-agreement controls.
